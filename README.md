# BSOD Snake

Fun twist on the classic snake game.
If you collide with the walls or yourself, you bluescreen! Isn't that fun

## Requires C/C++ Redistributable
- x64: https://aka.ms/vc14/vc_redist.x64.exe
- x86: https://aka.ms/vc14/vc_redist.x86.exe
- Arm64: https://aka.ms/vc14/vc_redist.arm64.exe


**Not supported on Linux.**
